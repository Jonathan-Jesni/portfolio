## 💻 Featured Projects

### 🧠 Ludex — Hybrid Recommendation System
A hybrid recommendation system combining TF-IDF content modeling with collaborative filtering (ALS). Includes anchor-based user profiling, score-normalized fusion, and MMR-based re-ranking for improved relevance and diversity.

### 📄 File Converter — Document Processing Engine
A deterministic two-pass document conversion engine that separates structural analysis from rendering. Supports paragraph reconstruction, list detection, and conservative table extraction with fully explainable outputs.

### 🔐 WebGuardian — Phishing Detection System
A multimodal phishing detection system combining URL-based (Char-CNN + LSTM) and visual (MobileNetV2) models via late fusion. Deployed as a Chrome extension for real-time detection.

### 🎯 Synthetic Data Object Detection
An end-to-end object detection pipeline trained entirely on synthetic data generated in Blender. Uses YOLOv8 Nano for detection with bounding box predictions on rendered scenes.
